# Jason Holt - Interactive Resume

A modern, AI-enhanced interactive resume showcasing a hybrid skill set in frontend engineering and learning product design.

## Features

- **AI Fit Analyzer**: Powered by Gemini AI, analyzes company descriptions and explains why I'm a great match
- **Skills Visualization**: Interactive radar chart displaying cross-functional expertise
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Dark Theme**: Modern glassmorphism UI with smooth animations

## Tech Stack

- HTML5 / Tailwind CSS (CDN)
- Chart.js for data visualization
- Google Gemini AI API (locked)
- GitHub Pages hosting

## Local Development

```bash
python3 -m http.server 8000
```

Visit `http://localhost:8000` to preview.

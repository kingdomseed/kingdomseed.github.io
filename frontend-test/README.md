# Jason Holt - Personal Resume Website

A visually striking personal resume website built with HTML, Tailwind CSS, and JavaScript.

## Project Structure

- `index.html`: Main content file.
- `styles.css`: Custom styles and Tailwind directives.
- `main.js`: Interactive logic (Dark mode, Mobile menu).
- `tailwind.config.js`: Tailwind configuration.

## Features

- **Distinctive Design**: Charcoal + Neon Green palette.
- **Responsive Layout**: Asymmetric 60/40 split design.
- **Dark Mode**: Built-in toggle with local storage persistence.
- **Performance**: Minimal dependencies, semantic HTML.

## How to Run

1. Simply open `index.html` in any modern web browser.
2. For development with live reload, use a local server (e.g., Live Server in VS Code).

## Customization

Edit `tailwind.config.js` to modify the color palette or font settings.
